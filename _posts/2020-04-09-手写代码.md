### 快速排序

```python
def quickSort(nums, start, end):
    if start >= end:
        return
    pivot = nums[start]
    left = start
    right = end
    
    while left < right:
        while left < right and nums[right] > pivot:
            right -= 1
        nums[left] = nums[right]
        while left < right and nums[left] <= pivot:
            left += 1
        nums[right] = nums[left]
    nums[right] = pivot
    
    quickSort(nums, start, left - 1)
    quickSort(nums, left + 1, end)
    return nums
```
